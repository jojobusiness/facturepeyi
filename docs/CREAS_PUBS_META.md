# BRIEF VISUELS (ce qu'il faut comme image/vidéo)

> Le visuel compte autant que le texte. 3 directions, du plus simple au plus fort. Format **carré 1080×1080** (feed) + **vertical 1080×1920** (stories/reels).

### Visuel 1 — Capture produit (le plus crédible)
- Screenshot du **dashboard FacturPeyi** ou d'une **facture générée** (avec une vraie mention « TVA non applicable, art. 294 » ou « TVA 8,5 % » bien visible).
- Surimpression texte : Angle A → « Factur-X conforme ✅ » · Angle B → « TVA Guyane 0 % · Octroi de mer ✅ ».

### Visuel 2 — Drapeaux territoires (identité locale forte)
- Drapeaux/cartes Guyane 🇬🇫 + Martinique + Guadeloupe + Réunion.
- Texte : « Conçu pour les pros d'ici ».
- Idéal pour l'angle B (fierté locale).
